TabLooper + AutoScroll

Rotación automática de pestañas con auto‑scroll y alertas para dashboards.

Descargar (Chrome Web Store) Instalar en modo desarrollador

Características

Rotación Automática

Rotación inteligente de pestañas con control de tiempo total por pestaña igual al intervalo configurado.

Auto‑Scroll Avanzado

Scroll automático con modos down/ping‑pong, emulación wheel/PageDown, selector de contenedor personalizable y traversal de Shadow DOM.

Sistema de Alertas SVG

Alertas visuales y sonoras por códigos de estado: 404 (azul), 500 (rojo), 401 (amarillo) basadas en consecutividad y proporción con overlay personalizable.

Análisis de Gráficos

Selector de Alert Target (recomendado: svg rect), análisis de N barras por gráfico y retorno al inicio con espera de 2 segundos.

Control Inteligente

Filtro por dominio, pausa automática por interacción del usuario y múltiples modos de loop para diferentes casos de uso.

Configuración Avanzada

Max Scroll Timeout, Scroll Step personalizable, modo smooth, selector CSS personalizado y configuración completa de alertas por series.

Cómo usar

1. Instalación en Modo Desarrollador

Navega a chrome://extensions, activa el "Developer mode" y usa "Load unpacked" para cargar la extensión desde la carpeta descargada.

2. Panel de Control

Usa el popup para controlar: Start/Pause/Stop, Next/Prev, configurar Interval, Scroll Duration, Direction (down/ping-pong), Loop Mode, Filter Domain y Pause on interaction.

3. Opciones Avanzadas

Configura Max Scroll Timeout, Scroll Step, modo Smooth, Custom Scroll Selector, Shadow DOM traversal, emulación wheel/PageDown y sistema completo de alertas: series mínimas, consecutividad, proporción, volumen, colores OK y barras por gráfico.

4. Atajos de Teclado

Utiliza los atajos predefinidos para controlar rápidamente la rotación y el scroll sin necesidad de abrir el popup.

Privacidad

No se recolectan ni transmiten datos; todo ocurre localmente

Tu privacidad es nuestra prioridad. TabLooper + AutoScroll funciona completamente offline y no envía información a servidores externos.

Permisos y Justificación

  • tabs: Para acceder y gestionar las pestañas del navegador
  • storage: Para guardar configuraciones del usuario
  • scripting: Para inyectar scripts de auto-scroll en las páginas
  • host_permissions <all_urls>: Para funcionar en todas las páginas web
